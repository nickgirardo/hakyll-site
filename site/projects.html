Context Play/ Pause

A simple WebExtension: This allows its user to play or pause a tab's video by an option in the tab's context menu. This was made before a change in Firefox which allowed users to control videos similarly with function keys. This significantly reduced the need for the WebExtension, but I still find myself using it from time to time.

Grid Tetris

Experimental implementation of classic video game Tetris using CSS Grid for all display.

Controls:

  • A/D: Move right and left
  • J/K: Rotate piece
  • S: Soft drop
  • W: Hard drop
  • Space: Hold piece

This game lacks mobile support as a keyboard is required.

Running out of Space

Action game created in 48 hours for Ludum Dare 42 competition. Competion theme: "Running out of Space." Navigate ship through waves of enemies and a shrinking arena.

Controls:

  • WASD: Move
  • Click: Fire

This game lacks mobile support as a keyboard and mouse are required.
